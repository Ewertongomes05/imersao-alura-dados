# 📊 Imersão Dados com Python - Alura
Repositório com análises, visualizações e códigos desenvolvidos durante a Imersão Dados com Python da Alura.

--------
## 📂 Conteúdo
1- Notebooks (.ipynb) com soluções e análises

2- Uso de bibliotecas como:
  * Pandas → manipulação e análise de dados
  * Streamlit → criação de Dashboards.
  * Matplotlib, Seaborn e Plotly → visualização gráfica.
  * PyCountry → tranformar as siglas dos países de ISO-2 para a ISO-3 e criar graficos de area.
  * plotly → Criar gráficos interativos e bonitos, que permitem zoom, hover (mostrar informações ao passar o mouse), seleção, filtros e muito mais, direto no navegador.
--------

# 🎯 Objetivo
Aprimorar conhecimentos em análise de dados, manipulação e tratamento dos dados, análise por meio de gráficos, utilização de diferentes tipos de gráficos e conversão de tipos de dados com o pandas.
